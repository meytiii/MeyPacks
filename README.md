# MeyPacks - Your Personal Portal Backpack

MeyPacks is a lightweight and powerful backpack plugin that gives every player a personal, persistent inventory that can be accessed anywhere, anytime. Designed to be simple for players and highly configurable for server administrators, MeyPacks is the perfect solution for extra storage.

## âœ¨ Features

*   **Persistent Storage:** Backpack contents are automatically saved when a player logs out or the server restarts.
*   **Custom Sizing:** Define different backpack sizes for different players or groups using permissions. From a single row up to a full double chest!
*   **Easy Access:** Open the backpack via a simple command or by clicking a custom, non-droppable item in your inventory.
*   **Admin Tools:** Server admins can view and modify the contents of any online player's backpack.
*   **Death Control:** Configure whether players keep their backpack items on death or drop them on the ground.
*   **Highly Performant:** Optimized to be lightweight and have a minimal impact on server performance.
